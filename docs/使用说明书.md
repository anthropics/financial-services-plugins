## 1. 工具简介

本工具是基于 [Anthropic Financial Services Plugins](https://github.com/anthropics/financial-services-plugins) 的增强版，新增了 **A股和港股免费数据源** 支持，并提供 **独立运行的Web应用**。

### 核心特点

- **完全免费**：所有数据源均为免费公开接口，无需付费订阅
- **覆盖全面**：支持A股（上交所/深交所）和港股（港交所）
- **多端运行**：支持Windows电脑、手机浏览器、网页版
- **不依赖环境**：提供一键启动脚本，自动安装依赖
- **中文界面**：完整的中文用户界面和分析报告

### 两种使用方式

| 使用方式 | 适合人群 | 说明 |
|----------|----------|------|
| **Claude插件方式** | Claude用户 | 在Claude中通过命令调用，获得AI驱动的深度分析 |
| **独立Web应用** | 所有用户 | 独立运行的网页应用，无需Claude，直接查看数据 |

## 3. 安装指南

### 3.1 系统要求

- **操作系统**：Windows 10/11、macOS 10.15+、Linux
- **Python**：3.8 或更高版本
- **磁盘空间**：约 500MB
- **网络**：需要互联网连接（获取实时数据）

### 3.2 Windows 安装步骤

#### 第一步：安装 Python

1. 打开 https://www.python.org/downloads/
2. 下载最新版 Python（3.10+ 推荐）
3. 运行安装程序
4. **重要**：勾选 ✅ "Add Python to PATH"
5. 点击 "Install Now"

#### 第二步：下载本工具

**方式一：Git 克隆（推荐）**
```bash
git clone https://github.com/anthropics/financial-services-plugins.git
cd financial-services-plugins/standalone-app
```

**方式二：直接下载**
1. 打开 GitHub 页面
2. 点击 "Code" → "Download ZIP"
3. 解压到任意目录
4. 进入 `standalone-app` 文件夹

#### 第三步：启动应用

**方式一：双击 start.bat（最简单）**
- 双击 `start.bat` 文件
- 首次运行会自动安装依赖（约2-5分钟）
- 安装完成后自动打开浏览器

**方式二：命令行启动**
```bash
cd standalone-app
pip install -r requirements.txt
python app.py
```

### 3.3 Mac/Linux 安装步骤

```bash
# 克隆仓库
git clone https://github.com/anthropics/financial-services-plugins.git
cd financial-services-plugins/standalone-app

# 安装依赖
pip3 install -r requirements.txt

# 启动
python3 app.py
# 或使用启动脚本
chmod +x start.sh
./start.sh
```

### 3.4 手机访问

1. 确保手机和电脑在同一个 WiFi 网络下
2. 在电脑上启动应用：`python app.py --host 0.0.0.0`
3. 查看电脑的 IP 地址：
   - Windows：打开命令提示符，输入 `ipconfig`，找到 IPv4 地址
   - Mac/Linux：终端输入 `ifconfig` 或 `ip addr`
4. 手机浏览器访问：`http://电脑IP地址:8888`
5. 例如电脑 IP 是 `192.168.1.100`，则访问 `http://192.168.1.100:8888`

### 3.5 打包为 Windows 可执行文件（免安装版）

如果希望生成一个不需要安装 Python 的 .exe 文件：

```bash
cd standalone-app
pip install pyinstaller
pyinstaller build_windows.spec --clean
```

打包完成后，在 `dist/中国股市分析工具/` 目录下找到可执行文件，可以直接分发给其他人使用。

## 4. 使用教程

### 4.1 市场概览

1. 打开应用后默认显示"市场概览"页面
2. 点击"刷新数据"按钮获取最新的指数行情
3. 可以看到上证指数、深证成指、创业板指等主要指数
4. 点击"加载板块数据"查看行业板块涨跌排行

**展示内容：**
- 主要指数最新价格和涨跌幅
- 北向资金净流入金额
- 行业板块涨跌排行榜（领涨TOP15 + 领跌TOP5）

### 4.2 A股查询

1. 点击导航栏"A股查询"
2. 在搜索框输入股票代码（如 `600519`）或名称
3. 点击"查询"或按回车
4. 也可以点击下方的快捷按钮直接查询热门股票

**查询结果包含：**
- 实时价格、涨跌幅、成交额
- 市盈率(PE)、市净率(PB)
- 总市值、流通市值
- 换手率、量比、振幅
- 60日涨跌幅、年初至今涨跌幅
- 近半年日K线数据（最近20个交易日）
- 财务数据摘要

### 4.3 港股查询

1. 点击导航栏"港股查询"
2. 输入港股代码（如 `00700` 代表腾讯控股）
3. 点击"查询"

**港股代码说明：**
- 港股代码为4-5位数字，前面补零
- 例如：腾讯 = 00700，阿里巴巴 = 09988
- 汇丰 = 00005，中国平安 = 02318

**查询结果包含：**
- 实时价格（港币）、涨跌幅
- 市盈率、市净率、股息率
- 52周最高/最低
- Beta系数
- 近半年历史行情

### 4.4 股票筛选

1. 点击导航栏"股票筛选"
2. 设置筛选条件（可选）：
   - PE范围（市盈率）
   - PB上限（市净率）
   - 市值范围（亿元）
3. 或点击预设策略按钮快速设置：
   - **价值型**：PE<15, PB<2, 市值>50亿
   - **成长型**：PE<50, 市值>30亿
   - **高股息**：PE<20, PB<3, 市值>100亿
   - **小盘股**：PE<40, 市值20-100亿
4. 点击"开始筛选"

**筛选结果：**
- 返回符合条件的前30只股票
- 显示代码、名称、价格、涨跌幅、PE、PB、市值
- 点击代码可跳转到A股详情页

### 4.5 AH对比

1. 点击导航栏"AH对比"
2. 点击"加载AH股数据"
3. 查看同时在A股和港股上市的公司列表
4. 关注AH比价列：
   - 比价 > 1.3：A股显著溢价
   - 比价 ≈ 1.0：两地估值接近
   - 比价 < 1.0：H股溢价（罕见）

## 5. 数据源说明

本工具使用的所有数据源均为 **完全免费** 的公开接口：

### AKShare（核心数据源）

- **官网**：https://akshare.akfamily.xyz/
- **许可**：MIT 开源许可
- **覆盖**：A股实时行情、历史数据、财务报表、行业板块、资金流向、北向资金、融资融券等
- **特点**：国内最全面的免费金融数据接口

### Yahoo Finance（港股数据源）

- **接口**：yfinance Python库
- **覆盖**：港股/美股实时行情、历史数据、财务报表、分析师评级
- **特点**：国际通用的免费金融数据源

### BaoStock（历史数据补充）

- **官网**：http://baostock.com/
- **许可**：免费开源
- **覆盖**：A股历史行情（日/周/月K线）、财务指标
- **特点**：数据质量高，支持复权

### 数据延迟说明

| 数据类型 | 延迟 | 说明 |
|----------|------|------|
| A股实时行情 | 约15-30秒 | 非实时推送，每次请求获取最新数据 |
| 港股实时行情 | 约15分钟 | Yahoo Finance免费版有延迟 |
| 历史行情 | 收盘后更新 | 每个交易日收盘后更新 |
| 财务数据 | 报告期后 | 跟随上市公司公告时间 |

## 6. Claude插件使用

### 6.1 安装插件

**在 Claude Code 中：**
```bash
# 添加市场
claude plugin marketplace add anthropics/financial-services-plugins

# 安装核心插件（必须先装）
claude plugin install financial-analysis@financial-services-plugins

# 安装中国市场插件
claude plugin install chinese-markets@financial-services-plugins
```

**在 Claude Cowork 中：**
从插件市场直接安装。

### 6.2 使用命令

安装后，可以在Claude中使用以下命令：

```
/a-stock 600519          # 分析贵州茅台
/hk-stock 00700          # 分析腾讯控股
/market-overview          # 查看今日市场概览
/stock-compare 600519,000858  # 对比茅台和五粮液
/dcf-cn 600519            # 茅台DCF估值
/screener PE<20 ROE>15%   # 筛选低估值高ROE股票
/financial-report 600519 2024年报  # 茅台年报分析
```

### 6.3 技能自动触发

安装插件后，当你在Claude中提到以下内容时，相关技能会自动激活：

- "帮我分析600519" → 触发 a-stock-analysis 技能
- "港股00700怎么样" → 触发 hk-stock-analysis 技能
- "今天市场怎么样" → 触发 cn-market-overview 技能
- "帮我看看茅台的财报" → 触发 cn-financial-report 技能
- "茅台现在估值如何" → 触发 cn-valuation 技能

## 7. 独立应用使用

### 7.1 启动方式

**Windows：**
```
双击 start.bat
```

**命令行：**
```bash
# 基本启动（仅本机访问）
python app.py

# 允许局域网访问（手机可用）
python app.py --host 0.0.0.0

# 指定端口
python app.py --port 9999

# 生产模式
python app.py --production --host 0.0.0.0

# 不自动打开浏览器
python app.py --no-browser
```

### 7.2 各平台使用方式

#### Windows 电脑

1. 双击 `start.bat`
2. 等待浏览器自动打开
3. 使用各功能模块

#### 手机浏览器

1. 电脑启动：`python app.py --host 0.0.0.0`
2. 查看电脑IP（命令提示符输入 `ipconfig`）
3. 手机浏览器访问 `http://电脑IP:8888`
4. 界面自动适配手机屏幕

#### 网页版（部署到服务器）

1. 将 `standalone-app` 目录上传到服务器
2. 安装依赖：`pip install -r requirements.txt`
3. 启动：`python app.py --host 0.0.0.0 --production`
4. 配置反向代理（nginx）指向 8888 端口
5. 通过域名或服务器IP访问

### 7.3 API 接口

独立应用提供 RESTful API，可供其他程序调用：

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/a-stock/realtime/{代码}` | GET | A股实时行情 |
| `/api/a-stock/history/{代码}` | GET | A股历史行情 |
| `/api/a-stock/financial/{代码}` | GET | A股财务数据 |
| `/api/hk-stock/realtime/{代码}` | GET | 港股实时行情 |
| `/api/hk-stock/history/{代码}` | GET | 港股历史行情 |
| `/api/hk-stock/financial/{代码}` | GET | 港股财务数据 |
| `/api/market/overview` | GET | 市场概览 |
| `/api/market/industries` | GET | 行业板块 |
| `/api/market/ah-comparison` | GET | AH股对比 |
| `/api/screener` | POST | 股票筛选 |
| `/api/status` | GET | 系统状态 |

**示例：**
```bash
# 获取贵州茅台实时行情
curl http://localhost:8888/api/a-stock/realtime/600519

# 获取腾讯港股数据
curl http://localhost:8888/api/hk-stock/realtime/00700

# 股票筛选
curl -X POST http://localhost:8888/api/screener \
  -H "Content-Type: application/json" \
  -d '{"pe_max": "20", "mv_min": "100"}'
```

## 8. 常见问题

### Q1: 启动失败，提示"未检测到Python"

**解决方案：**
1. 确认已安装 Python 3.8+
2. 安装时必须勾选"Add Python to PATH"
3. 安装后重启命令提示符
4. 运行 `python --version` 确认

### Q2: 安装依赖时报错

**解决方案：**
```bash
# 使用国内镜像源
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 或使用阿里云镜像
pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
```

### Q3: 数据获取失败

**可能原因：**
- 网络连接问题
- 非交易时间（部分实时数据不可用）
- 数据源接口临时维护

**解决方案：**
- 检查网络连接
- 交易日交易时间段内查询效果最佳
- 稍后重试

### Q4: 手机无法访问

**解决方案：**
1. 确保使用 `--host 0.0.0.0` 参数启动
2. 确保手机和电脑在同一WiFi网络
3. 检查电脑防火墙是否阻止了8888端口
4. Windows防火墙设置：允许Python通过防火墙

### Q5: 港股数据显示为空

**可能原因：**
- Yahoo Finance 在中国大陆可能需要网络代理
- 港股代码格式不正确

**解决方案：**
- 确保可以访问 Yahoo Finance
- 港股代码格式：4-5位数字（如00700，不需要.HK后缀）

### Q6: 如何修改端口？

```bash
python app.py --port 9999
```

### Q7: 如何在后台运行？

**Linux/Mac：**
```bash
nohup python3 app.py --host 0.0.0.0 --production &
```

**Windows：**
使用 start.bat 或创建 Windows 服务。
