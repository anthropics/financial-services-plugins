# 中国市场估值分析技能

## 一、DCF估值（现金流折现法）

### 中国市场WACC计算

**股权成本（CAPM模型）：**
```
Ke = Rf + β × ERP + CRP

其中：
Rf = 中国10年期国债收益率（约2.0-3.0%）
β = 个股相对沪深300的Beta系数
ERP = 股权风险溢价（A股约6.0-7.0%）
CRP = 国家风险溢价（对于港股外资，约1.0-2.0%）
```

**债务成本：**
```
Kd = 企业贷款利率 × (1 - 所得税率)

参考：
• LPR 1年期：约3.1%
• LPR 5年期以上：约3.6%
• 企业实际借款利率：LPR + 上浮（0-200bps）
• 所得税率：一般25%，高新技术企业15%
```

**典型WACC范围（A股）：**
- 大盘蓝筹：7-9%
- 中盘成长：9-12%
- 小盘高成长：12-15%

### 终值计算

**永续增长率选择（中国市场）：**
- 保守：2.0-2.5%（低于GDP目标增速）
- 中性：2.5-3.5%（接近长期通胀目标+实际增长）
- 乐观：3.5-5.0%（仅适用于行业龙头）

### 中国市场DCF特别调整

- **A股壳价值**：小市值公司可能有壳价值溢价（注册制后逐渐减弱）
- **港股折价**：H股相比A股通常有10-30%折价
- **VIE结构风险**：部分中概/港股互联网公司需考虑VIE结构风险折价
- **政策风险折价**：受监管影响大的行业（教育、游戏、地产）需额外风险溢价

## 二、相对估值

### PE估值法

```python
import akshare as ak

# 获取历史PE数据
stock_a_indicator_lg = ak.stock_a_indicator_lg(symbol="600519")
# 包含：pe, pe_ttm, pb, ps, ps_ttm, dv_ratio, dv_ttm, total_mv

# 计算PE百分位
pe_data = stock_a_indicator_lg['pe_ttm'].dropna()
current_pe = pe_data.iloc[-1]
percentile = (pe_data < current_pe).sum() / len(pe_data) * 100
```

**PE估值解读：**
```
估值百分位    评价        操作建议
< 10%        极度低估    考虑逐步建仓
10-30%       相对低估    可以关注
30-70%       合理区间    持有为主
70-90%       相对高估    谨慎追高
> 90%        极度高估    考虑减仓
```

### PB估值法

适用行业：银行、保险、券商、地产

```
PB < 1.0: 破净，可能低估（或资产质量差）
PB 1.0-2.0: 合理区间
PB > 3.0: 通常需要高ROE支撑
```

### PS估值法

适用行业：互联网、SaaS、高成长亏损企业

### PEG估值法

```
PEG = PE(TTM) / 未来3年预期净利润CAGR

PEG < 0.5: 严重低估
PEG 0.5-1.0: 合理偏低
PEG 1.0-1.5: 合理
PEG > 2.0: 可能高估
```

## 三、行业特殊估值方法

**银行：**
- PB估值为主，参考ROE-PB回归
- 拨备覆盖率调整后的PB

**地产：**
- NAV（净资产价值法）
- 土地储备价值重估

**消费品：**
- PE为主，关注确定性溢价
- 品牌价值估值

**科技/互联网：**
- PS 或 EV/Revenue
- 用户数 × 单用户价值
- SoTP（分部估值加总法）

**新能源/半导体：**
- PEG，关注行业景气周期位置
- 产能价值法
