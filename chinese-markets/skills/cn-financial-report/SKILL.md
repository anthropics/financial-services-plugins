# 中国上市公司财报分析技能

## 分析框架

### 1. 利润表分析

**营收质量评估：**
```
• 营收增速 vs 行业平均增速
• 营收集中度（前五大客户占比）
• 营收季节性分析（各季度占比）
• 合同负债/预收账款变化（前瞻指标）
```

**盈利能力拆解：**
```
营收 100%
├── 营业成本 -XX%
│   └── 毛利率 = XX%（关注趋势）
├── 销售费用 -XX%（关注占比变化）
├── 管理费用 -XX%
├── 研发费用 -XX%（资本化比例）
├── 财务费用 -XX%（利息收支）
├── 资产减值 -XX%（关注异常）
└── 净利率 = XX%
```

### 2. 资产负债表分析

**资产质量：**
- 货币资金/短期借款覆盖率
- 应收账款周转天数（vs 上期和行业）
- 存货周转天数（vs 上期和行业）
- 商誉/净资产比（> 30% 需警惕）
- 固定资产更新率

**负债结构：**
- 有息负债率（有息负债/总资产）
- 短期偿债能力：流动比率、速动比率
- 长期偿债能力：资产负债率、利息覆盖倍数

### 3. 现金流量表分析

**现金流质量评估：**
```
✅ 健康模式：经营(+) 投资(-) 筹资(-)
   → 主业赚钱，积极投资，减少负债

⚠️ 扩张模式：经营(+) 投资(-) 筹资(+)
   → 主业赚钱，大举投资，融资支撑

🚨 危险模式：经营(-) 投资(-) 筹资(+)
   → 主业亏损，仍在投资，靠融资维持

❌ 衰退模式：经营(-) 投资(+) 筹资(-)
   → 主业亏损，变卖资产，偿还债务
```

**核心指标：**
- 经营现金流/净利润 > 1.0（盈利质量好）
- 自由现金流 = 经营现金流 - 资本支出
- 现金分红/自由现金流（分红可持续性）

### 4. 杜邦分析（ROE拆解）

```
ROE = 净利率 × 资产周转率 × 权益乘数

净利率 ↑ → 盈利能力提升
资产周转率 ↑ → 运营效率提升
权益乘数 ↑ → 杠杆提升（需关注风险）

三种ROE提升路径：
1. 高净利率型（茅台模式）：毛利率极高，品牌溢价
2. 高周转型（沃尔玛模式）：薄利多销，效率为王
3. 高杠杆型（银行模式）：低利差但高杠杆
```

### 5. 财务预警信号

**红色预警（高风险）：**
- 🚨 扣非净利润连续2年亏损
- 🚨 经营现金流连续为负
- 🚨 应收账款增速 > 营收增速 × 2
- 🚨 大额商誉 + 业绩不达预期
- 🚨 大股东高比例质押

**黄色预警（需关注）：**
- ⚠️ 毛利率连续3个季度下降
- ⚠️ 存货周转天数显著增加
- ⚠️ 预付款项异常增加
- ⚠️ 其他应收款大幅增长
- ⚠️ 研发资本化比例过高
