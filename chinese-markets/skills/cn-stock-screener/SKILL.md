# 股票筛选技能

## 数据获取

```python
import akshare as ak

# A股全市场数据
stock_zh_a_spot_em = ak.stock_zh_a_spot_em()
# 包含: 代码,名称,最新价,涨跌幅,涨跌额,成交量,成交额,振幅,最高,最低,今开,昨收,
#       量比,换手率,市盈率-动态,市净率,总市值,流通市值,涨速,5分钟涨跌,60日涨跌幅,年初至今涨跌幅

# 港股全市场数据
stock_hk_spot_em = ak.stock_hk_spot_em()

# 个股财务指标（需逐个获取或使用批量接口）
stock_financial_abstract_ths = ak.stock_financial_abstract_ths(symbol="600519", indicator="按年度")
```

## 筛选引擎

### 支持的筛选条件

**估值指标：**
| 指标 | 参数名 | 示例 |
|------|--------|------|
| 市盈率(动态) | PE | PE<20, PE>10 |
| 市净率 | PB | PB<3 |
| 市销率 | PS | PS<5 |
| 股息率 | DY | DY>3% |

**规模指标：**
| 指标 | 参数名 | 示例 |
|------|--------|------|
| 总市值(亿) | MV | MV>100, MV<500 |
| 流通市值(亿) | FMV | FMV>50 |

**行情指标：**
| 指标 | 参数名 | 示例 |
|------|--------|------|
| 涨跌幅(%) | CHG | CHG>5% |
| 换手率(%) | TR | TR>3% |
| 量比 | VR | VR>2 |
| 60日涨跌幅 | CHG60 | CHG60>20% |
| 年初至今 | YTD | YTD>30% |

**财务指标（需额外获取）：**
| 指标 | 参数名 | 示例 |
|------|--------|------|
| ROE | ROE | ROE>15% |
| 毛利率 | GPM | GPM>30% |
| 净利率 | NPM | NPM>10% |
| 营收增速 | RG | RG>20% |
| 净利增速 | NPG | NPG>15% |
| 资产负债率 | DAR | DAR<60% |

## 预设策略模板

### 1. 格雷厄姆价值策略
```
PE < 15
PB < 1.5
PE × PB < 22.5
DY > 2%
DAR < 50%
MV > 50亿
```

### 2. 彼得·林奇成长策略
```
PEG < 1.0
RG > 20%
NPG > 20%
ROE > 15%
DAR < 60%
```

### 3. 巴菲特护城河策略
```
ROE > 20%（连续3年）
GPM > 40%
NPM > 15%
DAR < 40%
MV > 200亿
```

### 4. 高股息策略
```
DY > 4%
PE < 20
连续3年分红
经营现金流为正
DAR < 60%
```

### 5. 小盘成长策略
```
MV > 30亿, MV < 100亿
RG > 30%
NPG > 30%
ROE > 12%
```

### 6. 困境反转策略
```
PE < 行业中位数
PB < 2
近1年跌幅 > 30%
最新季度营收或利润改善
```

## 排序规则

默认排序：
1. 总市值降序（大到小）

可选排序：
- PE升序（便宜到贵）
- ROE降序（高到低）
- 股息率降序
- 营收增速降序

## 输出限制

- 默认返回前30只符合条件的股票
- 最多返回50只
- 每只股票显示：代码、名称、行业、市值、PE、PB、ROE、股息率

## 免责声明

筛选结果仅基于历史和当前数据，不构成投资建议。过去的财务表现不代表未来。
