# 港股分析命令

查询和分析港股上市公司的实时行情、财务报表和估值指标。

## 工作流程

### 第一步：获取信息

解析输入内容：
- 港股代码（4-5位数字，如 00700）或股票名称
- 港股代码格式：数字前补零至5位，如 700 → 00700

如未提供，请询问：
- "请提供要分析的港股代码或名称"

### 第二步：数据获取

使用免费数据源获取以下信息：

1. **实时行情数据**（Yahoo Finance / AKShare）：
   - 最新价（港币）、涨跌幅、成交量
   - 最高价、最低价、开盘价、昨收价
   - 52周最高/最低、市值

2. **历史行情数据**（Yahoo Finance）：
   - 近 1 年日K线数据
   - 近 5 年周K线数据
   - 计算均线和技术指标

3. **基本面数据**（Yahoo Finance / AKShare）：
   - 财务报表（年报/中报）
   - 主要财务指标
   - 股息信息和派息历史

4. **AH股对比**（如适用）：
   - AH股溢价/折价率
   - 两地估值差异分析

### 第三步：生成分析报告

提供以下输出：

1. **个股概览**：
   - 公司基本信息
   - 当日行情摘要
   - 关键财务指标（以港币和人民币双币种展示）

2. **技术分析**：
   - 均线系统分析
   - 成交量趋势
   - 关键价位分析

3. **基本面分析**：
   - 盈利能力分析
   - 成长性评估
   - 财务稳健性

4. **估值分析**：
   - PE/PB/PS 当前与历史比较
   - 同行业港股对比
   - AH溢价分析（如为两地上市）

### 第四步：输出结果

提供：
1. **文字报告** - 完整分析
2. **估值表格** - 关键估值指标
3. **风险提示** - 含港股特有风险（汇率、流动性等）
