# 股票对比分析命令

对多只A股或港股进行横向财务指标和估值对比分析。

## 工作流程

### 第一步：获取信息

解析输入的股票列表：
- 支持逗号分隔的多个代码
- 支持混合A股和港股代码
- 建议 3-8 只股票为佳

如未提供，请询问：
- "请提供要对比的股票代码列表（逗号分隔）"
- "是否指定特定行业？"

### 第二步：数据获取

对每只股票获取：
1. 基本信息（名称、行业、市值）
2. 最新行情数据
3. 关键财务指标（近3年）
4. 估值指标

### 第三步：生成对比表

```
【可比公司分析】

┌──────────┬──────────┬──────────┬──────────┬──────────┐
│ 指标      │ 股票A    │ 股票B    │ 股票C    │ 中位数   │
├──────────┼──────────┼──────────┼──────────┼──────────┤
│ 市值(亿)  │ X,XXX    │ X,XXX    │ X,XXX    │ X,XXX    │
│ 营收(亿)  │ X,XXX    │ X,XXX    │ X,XXX    │ X,XXX    │
│ 营收增速   │ XX.X%    │ XX.X%    │ XX.X%    │ XX.X%    │
│ 净利润(亿) │ XXX      │ XXX      │ XXX      │ XXX      │
│ 净利增速   │ XX.X%    │ XX.X%    │ XX.X%    │ XX.X%    │
│ 毛利率     │ XX.X%    │ XX.X%    │ XX.X%    │ XX.X%    │
│ 净利率     │ XX.X%    │ XX.X%    │ XX.X%    │ XX.X%    │
│ ROE       │ XX.X%    │ XX.X%    │ XX.X%    │ XX.X%    │
│ PE(TTM)   │ XX.X     │ XX.X     │ XX.X     │ XX.X     │
│ PB        │ X.XX     │ X.XX     │ X.XX     │ X.XX     │
│ 股息率     │ X.XX%    │ X.XX%    │ X.XX%    │ X.XX%    │
└──────────┴──────────┴──────────┴──────────┴──────────┘
```

### 第四步：输出结论

- 各公司优劣势总结
- 估值相对高低分析
- 综合评价
