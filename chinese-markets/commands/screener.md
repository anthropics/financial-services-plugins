# 股票筛选器命令

根据多维度条件筛选A股和港股股票。

## 工作流程

### 第一步：解析筛选条件

支持的筛选维度：

**估值指标：**
- PE（市盈率）：PE<20, PE>10
- PB（市净率）：PB<3, PB>1
- PS（市销率）：PS<5
- 股息率：股息率>3%

**财务指标：**
- ROE：ROE>15%
- 毛利率：毛利率>30%
- 净利率：净利率>10%
- 营收增速：营收增速>20%
- 净利增速：净利增速>15%
- 资产负债率：资产负债率<60%

**规模指标：**
- 市值：市值>100亿, 市值<500亿
- 营收：营收>50亿

**市场筛选：**
- 市场：A股, 港股, 沪市, 深市, 创业板, 科创板
- 行业：白酒, 新能源, 半导体, 医药...

### 第二步：执行筛选

1. 获取全市场数据
2. 应用筛选条件
3. 按市值排序（可自定义）
4. 返回前20-50条结果

### 第三步：输出结果

```

├──────┼──────────┼──────┼──────┼──────┼──────┼────────┤
│ XXXXX│ XXXXXXXX │ XXXX │ XXX亿│ XX.X │ XX.X%│ X.XX%  │

```

## 预设筛选模板

- **价值型**：PE<15, PB<2, ROE>12%, 股息率>3%
- **成长型**：营收增速>30%, 净利增速>30%, ROE>15%
- **白马股**：市值>200亿, ROE>15%, PE<30, 连续5年盈利
- **高股息**：股息率>4%, 连续3年分红, PE<20
- **小盘成长**：市值<100亿, 营收增速>50%, 净利增速>50%
